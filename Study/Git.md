""""""" GIT """"""""


1. Git init: to initialize empty git repo.
2. Git status: to check if git is connected.
3. Git add .: To add the files to git.
4. git commit -m "msg": to commit the files and add message.
5. .gitkeep: when we want git to not ignore the empty files in our folders.
6. git stash: when we wanna store temporary data in our repo from others repo.
7. git pull: to get what has been committed to git and also to get data from other repo.(fetch+merge).
8. git log: All execution logs.
9. git clone repo link: to use the same repo to upload add new files without duplication.
10. git branch branch_name: create branch
10. git checkout -b "branch name": switch to it.
